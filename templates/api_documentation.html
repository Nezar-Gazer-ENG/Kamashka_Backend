Kamashka API Documentation

Note: All API endpoints return JSON responses unless otherwise specified.

Job Postings Endpoints

GET /api/job-postings/

Returns a list of all active job postings with pagination.

GET /api/job-postings/<id>/

Returns details of a specific job posting.

Blog Posts Endpoints

GET /api/blog-posts/

Returns a list of all published blog posts with pagination.

GET /api/blog-posts/<id>/

Returns details of a specific blog post.

Job Applications Endpoint

POST /api/job-application/

Submits a new job application. Requires multipart/form-data encoding.

Required fields: job_posting, full_name, email, phone, resume, consent_to_data_processing

Optional fields: nationality, country, city, cover_letter

File types allowed: PDF, DOC, DOCX (max 5MB)

Utility Endpoints

GET /api/resume/<application_id>/

Downloads a resume file for a specific application.
